# 问题记录

## ubuntu12如何安装vim 7.4

安装：
sudo add-apt-repository ppa:nmi/vim-snapshots
sudo apt-get update; sudo apt-get install vim
卸载还原：
sudo apt-get install ppa-purge; sudo ppa-purge ppa:nmi/vim-snapshots
